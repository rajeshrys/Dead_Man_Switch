# Cron Monitor CLI

A command-line tool to monitor backend cron jobs.

## Installation

```bash
npm install -g cron-monitor

Login

cron-monitor login YOUR_API_KEY

Sync Jobs
cron-monitor sync

View Logs
cron-monitor logs

Check Status
cron-monitor status

Start Monitoring
cron-monitor start

Help
cron-monitor --help

Version
cron-monitor --version
